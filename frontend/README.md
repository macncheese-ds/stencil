# Stencil Frontend

Vite + React + Tailwind skeleton for the Stencil control UI.

Quick start (PowerShell):

```powershell
cd c:\app\stencil\frontend
npm install
npm run dev
```

Set the API base URL using env variable when running Vite:
- Create `.env` in this folder with `VITE_API_BASE=http://localhost:4000/api`

This frontend provides a minimal login form and a grid of lines to Start/Stop/Reset.
